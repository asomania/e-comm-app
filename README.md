Projeyi clonladıktan sonra yapılması gerekenler :  <br/>
1. npm install ile npm bağımlılıkları indirilmeli. <br/>
2. cd backend ile composer install yapılmadı laravel bağımlılıkları için. <br/>
3. php artisan serve ile backend servisi kaldırılır. Servisteki url projemin içindeki api/index.ts ile urller eşleşmeli. <br/>
4. en son ana promizde front end npm run dev ile ayağa kaldırılmalıdır.
